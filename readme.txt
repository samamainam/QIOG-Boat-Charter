=== QIOG Boat Charter Builder ===

== Description ==
QIOG Boat Charter Builder is a complete charter booking solution with drag-and-drop stops, dynamic pricing, checkout pages, and automated email notifications.

== Features ==
- Drag & drop charter stops
- Dynamic pricing with upgrades
- Auto-generated checkout pages
- Customizable checkout URL
- Admin & customer email templates
- Elementor compatible
- Clean uninstall

== Installation ==
1. Upload the plugin zip
2. Activate the plugin
3. Go to Charter Customization
4. Set checkout URL
5. Done
